## ⭐ v8 — 표준 확정 (2026-05-11)

[**광주데시앙_X배너_v8.html**](광주데시앙_X배너_v8.html) — **현재 표준 모델**

**핵심 디자인 토큰** → [`어플_설계/X배너_표준_v8.json`](../어플_설계/X배너_표준_v8.json)

### 디자인 컨셉
- **컬러**: 코랄·크림 (입주 축하 따뜻함)
- **레이아웃**: 3그룹 구조 (헤더 / 매장+혜택메시지 / 표+결제카드)
- **그룹 간격**: 60px (그룹 내부 16~24px)

### 핵심 디자인 요소
- 배경 #FFF5EE 크림 / 텍스트 #2A2520 다크브라운 / 강조 #E55A3D 코랄
- "입주 특별 프로모션" — Pretendard 800, 65px, 코랄
- 매장명 — 위아래 1.5px 라인 + 35px 800
- "최대 109만원 특별 혜택" — 3줄 (35/40/35px), 109만원만 코랄
- 혜택표·결제 안내 — 둥근 14px 박스 + 그림자
- 결제 카드 3개 (POINT 01/02/03) — 옅은 브랜드 컬러 배경
- 한국어 자간 가이드 적용 (큰 글씨일수록 -0.04~-0.06em 음수)

## 🔄 어플 자동 생성 흐름 (향후 Phase 2)

```
[1] 품의서 PDF 업로드
        ↓
[2] 자동 추출 (동의어사전.json + 추출 로직)
    → 아파트명, 매장명, 기간, 혜택표 등 변수 추출
        ↓
[3] 톤 결정 (카피 뱅크 _자동_추천_규칙)
        ↓
[4] X배너 HTML 생성 (v8 템플릿 + 변수 치환)
    → X배너_표준_v8.json의 디자인 토큰 적용
    → 카피_뱅크.json의 톤별 카피 적용
    → 제품_라이브러리.json의 이미지 자동 매칭
        ↓
[5] PDF/PNG 내보내기 (Playwright)
    → 인쇄소 납품용 600×1800mm PDF
```
