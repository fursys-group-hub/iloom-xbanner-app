# 일룸 인쇄용 X배너 어플 기획서

> 입주박람회 시안을 외주 없이 자동으로 만들어주는 영업 담당자용 사내 도구

---

## 📌 한 줄 요약

품의서 PDF 하나만 올리면, **X배너·A4 POP·리플렛·단톡방 QR 시안** 등 박람회 인쇄물이 자동으로 나오는 어플이에요.

품의서를 9건 분석한 결과, 박람회마다 형식이 조금씩 다르지만 **공통 패턴이 명확**해서 자동화가 충분히 가능합니다.

---

## 🎯 왜 만드나요?

### 지금까지의 흐름 (불편한 점)
1. 입주박람회 결정 → 품의서 결재
2. 매번 PPTX 시안 요청서 작성
3. 외주업체에 메일 → 비용·시간 소요
4. 시안 받기 → 수정 요청 반복
5. 최종 인쇄

→ **박람회마다 같은 패턴 반복**. 변하는 건 아파트명·매장명·기간·혜택표뿐인데 매번 외주 비용 발생.

### 어플로 바뀌는 흐름
1. 품의서 결재 → **PDF 업로드**
2. 어플이 자동 추출 → 카피 톤만 선택
3. 미리보기 → **인쇄용 PDF 즉시 다운로드**

→ 외주 비용 절감 + 작업 시간 단축 (반나절 → 5분)

## 👥 누가 쓰나요?

### 주 사용자
- 일룸사업부 입주박람회 담당자
- 부서: 유통경쟁력강화팀(P-SHOP) / 리테일사업팀 — **2개 부서**
- ※ 구버전 품의서의 `직영사업팀(P-SHOP)`은 현재 유통경쟁력강화팀으로 통합됨

### 담당자 분석 (9건 샘플)
박람회 품의서를 작성하는 사람은 **9명 이상**으로 다양:
- 한나라, 이광호, 김단우, 이선재, 채선경, 최준민, 최지연, 김지수, 조아라 등

→ 어플은 **모든 작성자의 다양한 표 형식·문장 표현을 흡수**해야 함

### 사용 빈도
- 박람회 건마다 (월 2~5건 예상)
- 비개발자 → **클릭 몇 번으로 끝나야 함**

## ⚙️ 핵심 기능 5가지

### 1. 품의서 PDF 자동 추출
업로드한 PDF에서 다음 정보를 자동으로 뽑아냅니다.

| 추출 항목 | 예시 |
|---|---|
| 아파트명 | 광주 더 파크 비스타 데시앙 |
| 매장명·매장 수 | 일룸 분당서현(P) + 일룸 분당(B) |
| 박람회 기간·일수 | 2026/03/07~03/08 (2일) |
| 입주 시점 | 2026년 4월 |
| 세대수 | 1,690세대 |
| 객단가 | 300만원 |
| 혜택표 5구간 | 200·300·500·700·1000만원 × 혜택금액 |
| LG 연계 혜택 유무 | O / X |
| 특별 프로모션 종류 | 세트구매 / 옷장 / 모션 / 소파 / 학생방 / 단톡방 등 |
| 후기 이벤트 종류 | 네이버/카페/SNS/단톡방 |
| 단톡방 운영 여부 | O / X |
| 입주카페 URL | cafe.naver.com/... |
| 약명 | (광주데시앙) |

→ 사용자는 **확인하고 수정만** 하면 됨 (편집 가능)

### 2. 마케팅 카피 템플릿 뱅크
미리 만들어둔 카피 풀에서 자동 추천 + 수동 선택.

**3가지 톤:**
- **금액 강조**: "최대 109만원 특별 혜택, 지금 바로 만나 보세요!"
- **기간 강조**: "2일간의 특별한 혜택을 만나보세요"
- **축하 톤**: "광주 더 파크 비스타 데시앙 입주를 진심으로 축하드립니다"

**조건부 자동 추천 규칙:**
- 박람회 1~2일 → 금액 강조
- 박람회 3일 이상 → 기간 강조
- 최대 혜택 100만원 이상 → 헤드라인에 금액 자동
- 단톡방 운영 → 후기카피 자동 추가
- 박람회 미참여 → "매장 단독 혜택" 톤

### 3. 제품 이미지 라이브러리 + 자동 매칭
일룸 자주 쓰는 제품(약 30~40개) 사진을 사전 등록.

**카테고리별 (9건 분석 결과)**:
```
├─ 식탁: 업모션, 핀, 엘바패밀리, 듄
├─ 거실(소파): 밴쿠버, 어라운드, 스노즈컴팩트
├─ 거실장: 엘렌트
├─ 옷장/드레스룸: 컬렉트(키큰옷장), 테일러
├─ 침대: 토스티, 바젤, 헤이즐
├─ 매트리스: 헤이븐(슬림/디럭스/시그니처), 플렉서블
├─ 모션: 바젤 모션, 리빙 모션베드
├─ 책상: 업모션 테이블, 모션데스크, 로이 책상
└─ 학생방: 로이 책상세트, 로이 수납형 침대
```

기획서에서 제품명을 발견하면 → **라이브러리에서 자동 호출** → 시안 배치
→ "다른 사진으로 교체" 버튼으로 수동 보정 가능

### 4. 인쇄용 PDF 출력 (7종!)
**X배너만이 아닙니다. 박람회마다 필요한 인쇄물이 다양해요.**

| 출력물 | 사이즈 | 사용 케이스 |
|---|---|---|
| **X배너** | 가로 600 × 세로 1800mm | 박람회 부스 (가장 일반적) |
| **A4 POP** | 210 × 297mm | 매장 내 비치 |
| **3단 접지 리플렛** | A4 3등분 | 부산센텀, 디에이치처럼 정보량 많은 곳 |
| **단톡방 QR A4 시안** | 210 × 297mm | 한화포레나, 교대역모아 |
| **LG 연계 리플렛** | A4 또는 A5 | 수원성중흥처럼 LG와 공동제작 |
| **롤배너** | 미정 | 외부 박람회용 |
| **삽지** | A5/A6 | 디에이치처럼 신제품 강조용 |

→ 사용자가 체크박스로 필요한 것만 선택해서 한 번에 PDF 다운로드

### 5. 영업 담당 편의 기능
9개 품의서 분석에서 발견한 영업 담당의 실제 니즈:

| 기능 | 설명 |
|---|---|
| **매장별 즐겨찾기** | 분당서현, 평택점, 창원점 등 자주 쓰는 매장 미리 저장 |
| **이전 시안 불러오기** | "지난 평택 시안 그대로 + 아파트만 바꿔서" |
| **박람회 미참여 모드** | 수원성중흥처럼 박람회 없이 매장만 공략하는 케이스 → X배너 대신 리플렛 |
| **2아파트 동시 처리** | 잠실래미안+잠실르엘처럼 한 품의서에 2개 박람회 → 시안도 2종 자동 |
| **약명 자동 생성** | "광주 더 파크 비스타 데시앙" → "(광주데시앙)" |
| **매장 코드 자동** | (P)/(B)/(투자B) 자동 표기 |
| **자동 추출 실패 시 친절 안내** | "혜택표를 못 찾았어요 — 직접 입력해주세요" |
| **단톡방 QR 자동** | URL 입력 → QR 이미지 자동 생성 |
| **공동구매 단계 시각화** | 20/40/80/100세대 누적 그래프 자동 |
| **선착순 한정 강조** | "선착순 30개 한정" 자동 강조 박스 |

## 🛠️ 기술 스택 예상

### 어플 자동 동작 흐름

```
[1] PDF 업로드
    ↓
[2] 자동 추출 (아파트/매장/박람회/혜택표/특별프로모션/후기이벤트)
    ↓
[3] 케이스 자동 분류 (A/B/C/D/E/F)
    ↓
[4] 카피 자동 선택 (3가지 톤: 금액강조 / 기간강조 / 축하형)
    ↓
[5] 디자인 적용 (v8 표준 + 케이스별 변형)
    ↓
[6] 빈 공간 알고리즘 (콘텐츠 양에 따라 자동 조절)
    ↓
[7] 사용자 미리보기 + 토글로 조정
    ↓
[8] 인쇄용 PDF 다운로드
```

### 사용자가 토글로 조정할 수 있는 것

| 토글 | 선택지 | 설명 |

| 배경 이미지 | off / 침실 / 쿠시노 | 분위기 사진 깔기 |
