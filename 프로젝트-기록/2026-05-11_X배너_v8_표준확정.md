### 핵심 디자인 결정

1. **컬러 톤**: 코랄·크림 (입주 축하 따뜻함) — `_브랜드_자산` 톤 변경
   - 배경 #FFF5EE / 텍스트 #2A2520 / 강조 #E55A3D

2. **레이아웃**: 3그룹 구조
   - 그룹 1: 헤더 (welcome/아파트명/타이틀/기간)
   - 그룹 2: 매장 + 혜택 메시지
   - 그룹 3: 표 + 결제 안내
   - 그룹 간격 60px, 그룹 내부 16~24px

3. **타이포그래피**:
   - 한국어 자간 가이드 적용 (큰 글씨 -0.04~-0.06em)
   - 메인 타이틀 65px / 800 / 코랄
   - 109만원 메인 40px / 109만원만 코랄 900
   - 매장명 35px / 800 + 위아래 1.5px 라인

4. **박스 스타일**:
   - 혜택표·결제 안내: 14px 라운드 + 그림자
   - 매장명: 박스 X, 위아래 라인만
   - 카드 3개 (POINT 01/02/03): 옅은 브랜드 컬러 배경

5. **이미지 처리**:
   - 로고 PNG에 `mix-blend-mode: multiply` → 흰 배경 자연스럽게 흡수
   - SSGPAY는 SVG라 그대로

## 🤔 결정한 것

1. **광주 데시앙을 골든 스탠다드로 확정** — 외주업체 PNG가 검증 가능한 기준 + 5컬럼 신버전 + 금액강조 톤이라 가장 보편적 케이스

2. **컬러 톤은 코랄·크림으로** — 사용자가 4개 옵션 중 "입주 축하 따뜻함" 선택. 가구 박람회와 입주 환영 분위기에 가장 잘 맞음

3. **109 박스 → 라인 없는 3줄 텍스트로** — 매장명 박스에 라인이 있어 라인 중복 회피. 109만원 코랄 강조가 핵심 임팩트

4. **그룹 간격 60px** — 사용자가 76px → 60px로 줄여 조정. 콘텐츠 응집감 + 그룹 구분 둘 다 유지

5. **결제 안내는 카드 3개** — 사용자 제공 레퍼런스(브랜드 카드 3개)에 영감. 옅은 브랜드 컬러 + POINT 01/02/03 라벨 + 로고

6. **로고 PNG는 mix-blend-mode: multiply로 누끼 처리** — N pay, L.POINT, iloom 로고 모두 흰 배경 사라짐. CSS 한 줄로 모든 로고 통일 처리

7. **Pretendard 메인 폰트** — 한국어 모던 산세리프. Paybooc는 옵션으로 보존 (assets/fonts/)

## 💡 핵심 인사이트

1. **외주업체 PNG 1장이 진짜 골든 스탠다드** — 시안의 모든 디자인 결정의 기준점. 새로 만들 때마다 "원본과 비교"가 가장 효과적
2. **반복 사이클이 자산 검증에 효과적** — 카피 뱅크, 제품 라이브러리, 변수 시스템이 v1부터 v8까지 변화 없이 작동 → 자산 설계가 잘 됐다는 증거
3. **한국어 자간 가이드는 큰 글씨에서 결정적** — "광주" "더파크" 사이 간격이 자간 조정만으로 자연스러워짐
4. **그룹 시각화는 간격으로 결정됨** — 그룹 사이 60px, 그룹 내부 16~24px의 비율(약 3배)이 시각 묶음감의 핵심
5. **mix-blend-mode: multiply는 PNG 누끼의 마법** — 흰 배경 PNG 로고를 컬러 카드에 자연스럽게 흡수. CSS 한 줄로 모든 로고 통일
6. **사용자가 보낸 레퍼런스 이미지가 디자인 방향의 결정타** — "이런 느낌으로" 한 마디가 결제 안내 카드를 만들었음. 시각적 레퍼런스 ≫ 텍스트 설명
