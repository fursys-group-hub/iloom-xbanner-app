# 2026-05-29 — 특별 프로모션 배너별 카드형/행형 토글

## 배경
케이스 B "특별 프로모션 3종"이 2열 카드 그리드 고정이었음. 사용자가 "카드 말고 행으로 쌓고 싶다"며 방법을 물음 →
기존 "배너별 커스터마이즈(케이스 고정 금지)" 모델대로 **배너마다 카드형/행형을 고르는 토글**로 구현.

## ✅ 한 것
- **렌더**: `SpecialPromoGrid({title, cards, layout})`에 `layout`('card'/'row') 추가. 행형이면 `.promo-grid--rows` 클래스. `case-b/index.js`가 `layout: data._promoLayout` 전달.
- **banner.css** `.promo-grid--rows`: 1열 + `.promo-card`를 가로 flex(아이콘 정적·헤드라인 flex:1·설명 우측). 카드 색 블록 → 전체 폭 가로줄(더 촘촘·스캔 쉬움).
- **행형 다듬기(사용자 요청)**: 태그(시즌오프 스페셜 등) **숨김**, 헤드라인 **줄바꿈 없이 한 줄**(렌더에서 `\n`→공백, CSS `white-space:nowrap`) + **글씨 27px로 키움**.
- **편집창**(`renderPromoEditor`): 맨 위에 [카드형][행형] 세그먼트(`tone-group--2`). 클릭 → `state._promoLayout` 저장 + `renderPromoEditor()`(토글 활성 갱신) + `rerender()`. 결제 표시방식 토글과 동일 패턴.
- 기본값 = 카드형(`_promoLayout` 미설정). 배너별 저장(localStorage 드래프트). 미리보기·인쇄 PDF 공통.
