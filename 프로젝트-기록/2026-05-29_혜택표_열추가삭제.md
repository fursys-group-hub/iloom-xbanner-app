# 2026-05-29 — 혜택표 열 추가/삭제

## 배경
사용자: "품의서대로 표가 나왔는데, 품의서엔 없던 혜택이 하나 더 생기면 열을 추가해야 한다. 어떻게?"
표 렌더(`benefit-table.js`)는 이미 **동적 컬럼(4/5/6열)** 지원이었으나, **편집기엔 "+ 행 추가"만 있고 열 추가/삭제가 없었음**.

## ✅ 한 것
- **[+ 열 추가] 버튼**(index.html `#btnAddCol`, `+ 행 추가` 옆 `.te-add-actions`). 클릭 → 새 열 `{key:'colN', label:'새 혜택', style:'normal'}`을 **"총 혜택"(style:'total') 바로 앞**에 삽입 + 모든 행에 빈 값 추가. (총 혜택은 항상 맨 오른쪽 유지)
- **열 삭제**: 편집기 각 열 머리 아래 × 버튼(`.te-coldel`) 행 추가(`renderTableEditor`). 클릭 → 열 + 모든 행의 해당 key 삭제. **최소 2열 가드**(2열이면 토스트 "열은 최소 2개가 필요해요").
- 데이터 모델 그대로: `benefitTable.columns[{key,label,style}]` + `rows[{[key]:value}]`. 미리보기·인쇄 PDF 공통(렌더가 컬럼 동적).
