### 1. 기간 — 4단 폴백 전략
1. "기간" 라벨 → parsePeriodRange
2. 슬래시 풀데이트 "2026/07/31~08/02" (디에이치)
3. **박람회/1차 마커 날짜 줄 스캔** (납기/상시/운영/사전점검 제외)
   - 창원 한국어 "1차 26년 01월 17일(토)~18일(일)" → parsePeriodRange
   - 부산센텀 연도없는 "10월 11일(토)~10월 12일(일)" → `parseMonthDayRange` + 연도 보충(작성일/문서번호)
4. 박람회 미참여면 "운영 기간" 사용
- 결과: 8케이스 전부 기간 채움 (창원 2026-01-17~18 / 부산센텀 2025-10-11~12 / 수원성중흥 운영 2025-12-01~31 신규)

### 2. 박람회 미참여 감지
- "박람회 미참여 전략" 키워드 → `fields.noFair=true`
- to-app-state: noFair → maxBenefit "매장 단독 최대 N 혜택" / "지금 매장에서 만나보세요" (박람회 문구 금지)
- **분류 교정**: 수원성중흥이 후기 언급으로 case-d 오분류되던 것 → case-d 는 "6컬럼표 또는 2아파트" 시그니처 요구하도록 좁힘 → 수원성중흥 case-a 정상화

### 3. 세대수/객단가 본문 스캔 폴백
- 라벨이 빈 줄("세대수")·헤더("목표 객단가")를 잡으면 null → 본문에서 "N세대"(400↑) / "객단가 N만원" 스캔
- 디에이치(3064/320만)·대구범어(490/200만)·창원(967/260만) 복구

### 4. 매장 추출 3단 폴백
1. 공략매장 라벨 → parseStores('X점')
2. **따옴표 단독홍보** "'일룸 프리미엄샵 논현' …단독 홍보" → parseStoresFromQuotes
3. **번호 매긴 목록** "① 스타필드수원점 / 투자형B" "② 수원광교점 / 직영" → parseStoresNumbered (이름+코드)
- + 매장 2개↑면 "프리미엄샵" 접두어 제거
- 디에이치/잠실/**수원성중흥**(일룸 스타필드수원점·일룸 수원광교점) 자동
