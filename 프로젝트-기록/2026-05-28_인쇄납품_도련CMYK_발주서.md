# 2026-05-28 — 인쇄 납품 준비 (도련·CMYK + 발주서 + 출고 검수)

## 배경
"영업담당이 직접 만들어 인쇄업체에 납품" 관점에서 빠진 것 보강. 인쇄업체가 **CMYK + 도련**을 요구(사용자 확정). 우선 작업: 발주 사양서·이미지 해상도 경고·안전영역/도련 가이드.

## 한 것

### 1. 도련(bleed) + 재단선(crop marks) — 인쇄용 PDF (`app/server.js`)
- `captureBannerWith()` PDF 분기: 미디어 = 트림 600×1800 + 사방 **9px**(도련 3px + 재단선 여유). 배너를 9px 안쪽 배치, 바깥은 배너 배경색(크림)으로 채워 도련 처리. 트림 네 모서리에 L자 재단선(div) 추가.
- `page.pdf({ width:'618px', height:'1818px' })` 명시(= 163.7×481.2mm). 미리보기·PNG엔 영향 없음(캡처 분기 한정).
- **검증**: 출력 PDF 1페이지 463.9×1363.9pt = 163.7×481.2mm, 상단 모서리 재단선 + 크림 도련 육안 확인.

### 2. CMYK 변환 (`app/server.js`) — ★ 사용자 결정: **지금은 안 함(보류)**
- **결정(2026-05-28)**: CMYK 변환은 어플에서 하지 않기로. **출고본 = RGB + 도련 + 재단선 PDF**. 인쇄소가 RGB를 받아 자체 RIP 으로 처리하면 충분(이 인쇄소 기준). Ghostscript 설치 불필요.
- 코드는 **그대로 둠(비활성 폴백)** — gs 가 있으면 자동 CMYK, 없으면 RGB. 현재 gs 미설치라 항상 RGB 로 안전하게 동작. 향후 다른 인쇄소가 CMYK PDF 를 요구하면 ghostscript.com 설치 파일만 깔면 코드가 자동 인식.
  - `toCmyk(rgbBuffer)`(gs `-sColorConversionStrategy=CMYK`) + `findGhostscript()`(PATH→`C:\Program Files\gs\*\bin\gswin64c.exe` 탐지) + 응답 헤더 `X-Color-Mode`.
- 다운로드 안내: 현재 RGB 라 "색상 RGB" 안내가 뜨는데, 인쇄소가 RGB 수용하므로 정상. (안내 문구가 불필요하게 느껴지면 추후 톤 조정 가능.)

### 3. 인쇄 안전영역·도련 가이드 (미리보기 토글) — `index.html`/`editor.css`/`app.js`
- 툴바 "인쇄 가이드" 토글. `#bannerRoot` 안(배너 좌표계)에 `.print-guide` 오버레이 → 줌 transform 자동 추종, 출력엔 미포함.
- 구성: 점선 재단/안전선(`.pg-edge`, 12px 안쪽) + 상·하 거치대 가림 밴드(`.pg-top` 45px / `.pg-bot` 35px, 코랄 틴트 + "가려질 수 있어요").

### 4. 발주 사양서 자동 생성 — 완료 단계 (`index.html`/`app.js`)
- 자동: 제목(아파트+박람회), 크기 600×1800mm, 색상 CMYK, 도련 3mm, 형식 PDF(벡터).
- 입력: 수량·재질(드롭다운)·거치대 포함·납기일·담당자·연락처 → `state._order`(드래프트 저장).
- "발주서 복사"(클립보드) / ".txt 다운로드". 인쇄소 메일에 붙여 전달.

### 5. 이미지 해상도 경고 — `app/public/app.js`
- `recheckImages()` — 배너 내 `<img>`의 `naturalWidth` ÷ 표시폭(offsetWidth, 배너 600px=인쇄 600mm) → 인쇄 DPI 추정. **<100DPI면 경고**(이미지 로드 후 재검사). 완료 요약·다운로드 확인(`validationWarnings`)에 합류.

## 잔여 / 선택
- **CMYK 는 보류(사용자 결정)** — 현 인쇄소가 RGB 수용. 향후 CMYK PDF 요구하는 인쇄소를 쓰면 Ghostscript 설치만 하면 자동 동작(코드 완료, 인쇄소 ICC 프로파일 받으면 `-sOutputICCProfile` 로 코랄 정밀도↑).
- 도련 시 case-c 하단 사진은 크림 도련(사진 미확장) — 3px이라 미미하나 완전 bleed 원하면 배경 레이어 확장 필요.
- (병행) 사용자가 편집화면을 라이트/코랄 테마 + 스텝 UI(.stepper)로 리팩터링 중 — app.js 추가 편집은 충돌 피해 보류.
