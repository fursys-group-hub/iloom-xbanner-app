### 2. 입주박람회 시안 제작의 전체 흐름 파악

```
[1단계] 입주공략 결정
  → 품의서 작성 (한나라/이광호 등 담당자)
  → 결재 (안권희 / 한병화 / 정보은)

[2단계] 시안 요청
  → PPTX로 시안 요청서 작성
  → 시각물 결재 (사내 시스템)
  → 외주업체에 메일 전달

[3단계] 외주 작업
  → 디자이너가 X배너·A4 POP 제작
  → 회신 (이번 광주 데시앙 PNG가 그 결과물)

[4단계] 인쇄 → 박람회 현장 설치
```

### 3. 자동화 방향성 — 사용자(김수지)와 상의 후 결정

**핵심 인사이트:**
- 박람회마다 시안 디자인 구조는 거의 동일
- 변하는 건 아파트명·매장명·기간·혜택표·세트프로모션뿐
- 품의서에 위 모든 데이터가 텍스트로 들어있음 → 자동 추출 가능

**결정된 어플 구조:**

| 항목 | 결정 |
|---|---|
| 입력 | 품의서 PDF 1개 업로드 |
| 데이터 추출 | 자동 (PDF 텍스트 파싱) |
| 마케팅 카피 | 템플릿 뱅크 (3가지 톤) + 조건부 추천 |
| 제품 이미지 | **사내 라이브러리 사전 등록 → 기획서 제품명 자동 매칭** ← 옵션 A 선택 |
| 출력 | X배너 (600×1800mm) + A4 POP — PDF |
| 부가 옵션 | 단톡방 QR 시안 (토글) |

**카피 템플릿 — 3가지 톤 (실제 시안에서 추출):**
1. 금액 강조: "최대 109만원 특별 혜택, 지금 바로 만나 보세요!"
2. 기간 강조: "2일간의 특별한 혜택을 만나보세요"
3. 축하 톤: "○○ 입주를 진심으로 축하드립니다"
