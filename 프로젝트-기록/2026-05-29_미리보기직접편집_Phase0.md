## ✅ 오늘 한 것 (Phase 0 — 개편 토대)

  좌측 **검수 패널**(① 확인이 필요해요 ② 다듬기(배경 토글) ③ 인쇄용 내보내기+발주서) + 우측 **큰 미리보기**.

    안정적 클래스에 `data-edit`(패널명)·`data-field`(신뢰도 키) 부여. **렌더 컴포넌트는 안 건드림** → 시제품 픽셀·서버 export·검증 회귀 위험 0.
  - 영역 클릭 → `openEditor(panel, anchorEl)` 공통 **편집 팝오버**(영역 옆에 떠서 그 부분이 안 가려짐, 뷰포트 클램핑). 기존 폼 로직(혜택표 그리드·결제 체크박스/모드·프로모션 카드·`{강조}` 버튼)을 팝오버 패널로 이전(재작성 X, IDs·핸들러 보존).
  - Esc·바깥 클릭으로 닫힘. 다른 영역 클릭 시 패널 전환.
- **검수 패널 — 확인이 필요해요** — `state._extraction.fields` 의 medium/low/missing 항목(아파트명/기간/매장/혜택표/유의사항)을 리스트로.
  항목 클릭 → 미리보기 해당 위치로 스크롤+잠깐 강조(`is-target`)+편집 팝오버. "확인" 버튼 또는 직접 편집 시 `_reviewed` 기록 → 리스트·형광에서 제외(고치면 줄어듦).
- **미리보기 형광** — 저신뢰(needsReview) 영역에 코랄 점선(`is-flag`). 검토 완료 시 제거. (메모리 [[feedback_review_tool_direct_manipulation]] 등록)
