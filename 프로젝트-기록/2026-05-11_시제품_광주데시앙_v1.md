## 💡 핵심 인사이트

- **외주업체 PNG 한 장이 진짜 표준** — 그대로 따라가는 게 가장 빠르고 안전. 자체 디자인 만들지 않음.
- **인라인 CSS가 빠른 검증에 유리** — 시제품 단계에서는 CSS 분리/모듈화보다 단일 파일이 사용자가 바로 열어보고 비교 가능.
- **Pretendard CDN으로 한국어 폰트 즉시 해결** — `iloom 공식 폰트` 가이드 없이도 시제품 제작 가능. 디자인 시스템에 이미 명시된 폰트.
- **카피 뱅크의 변수 시스템이 검증됨** — JSON에 정의된 변수 10종이 실제 X배너에 매끄럽게 들어맞음. 향후 어플 자동 생성 시 변경 없이 활용 가능.
- **Playwright는 이미지 수집 + 렌더링 모두 활용** — 같은 의존성으로 두 가지 자동화 모두 가능. 어플의 핵심 도구.
