## 🤔 결정한 것 (규칙서 강화 후보)

### F. 알약 배지(기간) 패턴 — 다크 헤더 안에서
- 위치: 다크 섹션 타이틀 행 안에 `position: absolute; right: 14px` (우측 끝 고정)
- 스타일: `padding: 3px 12px 2px` / `background: rgba(255,255,255,0.18)` / `border-radius: 9999px`
- 폰트: 13px / weight 700 / -0.01em / `font-feature-settings: "tnum"` (숫자 정렬)
- 타이틀은 `text-align: center` 유지 → 시각적으로는 타이틀 가운데, 배지 우측 끝
- 부모는 `position: relative` 필수

### G. 날짜 표기 변형 (케이스별)
| 케이스 | 헤더 아래 행사기간 | 추가혜택 옆 배지 |
|--------|------------------|----------------|
| v2 (양쪽 다 표기) | `2026.05.22. ~ 2026.05.24.` (X배너용 풀 포맷) | `05. 22. ~ 06. 30.` (월일만) |
| v3 (전부 제거) | ❌ 제거 | ❌ 제거 |

- 헤더 보강 룰: `.period` 제거 시 `.main-title margin-bottom: 14px → 32px` (시각 균형)

### H. 주의사항 위치 조정 — padding-bottom 활용
- 기존: `.notices { flex: 1; justify-content: flex-end }` → 페이지 최하단 부착
- 개선: `padding-bottom: 32px` 추가 → 바닥 여백 확보, 시각적으로 위로 올라옴
- `flex: 1` 유지 → 본문과 주의사항 사이 빈 공간은 여전히 자동 흡수
- 푸터 로고는 `.notices` 외부의 별개 sibling → 영향 없음

### I. A4 시안 매장명 크기 가이드
- 1매장 / 2매장 케이스 모두 한 줄 가로 배치 (`|` 구분자)
- 크기 변경 이력 (이번 시안): **24px → 20px → 26px**
- 결론: A4 시안의 매장명은 **22~26px / weight 700** 범위에서 케이스별 조정 가능 (헤더 apt-name 45px과 충돌 안 나는 선)
