# 일룸 인쇄용 X배너 어플

품의서 PDF를 올리면 → 내용을 자동으로 읽어 → X배너(인쇄용 세로 배너) 시안을 만들고 → PNG/PDF로 내려받는 도구입니다.

품의서 한 장만 넣으면 매장명·혜택·제품 정보가 자동으로 채워진 초안이 만들어지고, 미리보기 화면에서 글자·이미지·영역 순서를 직접 손봐서 완성합니다.

## 무엇으로 만들어졌나요

- **Express** (웹서버 — 요청을 받고 화면을 돌려주는 역할)
- **pdfjs-dist** (PDF 글자 읽기 — 품의서 PDF에서 텍스트를 뽑아냄)
- **Playwright** (화면을 그대로 이미지/PDF로 찍어주는 도구 — 인쇄용 출력 생성)
- **Pretendard** (한글 폰트)

## 실행 방법

처음 한 번만 (필요한 부품 설치):

```bash
cd app
npm install
```

이후 실행:

```bash
cd app
npm start
```

실행되면 브라우저에서 아래 주소로 접속합니다:

```
http://localhost:3000
```

> Windows에서는 프로젝트 폴더의 `실행.bat` 파일을 더블클릭해도 실행됩니다.

## 폴더 구조

```
app/                앱 본체 (웹서버)
  ├─ server.js      서버 시작 파일
  ├─ public/        화면(HTML/CSS/JS) — 미리보기·편집 화면
  ├─ extract/       품의서 PDF에서 내용을 읽어내는 코드
  ├─ render/        배너를 그려내는 코드
  └─ scripts/       이미지 동기화·검증용 보조 스크립트
assets/             폰트·제품 이미지·배경·로고 등 디자인 재료
scripts/            제품 이미지 자동 수집 스크립트
기획서.md           프로젝트 기획 문서
프로젝트-기록/      작업 일지
```
